# Migração de Dados com SQL Server

Projeto simulando migração de dados de um sistema legado para um novo sistema.

## Tecnologias
- SQL Server
- T-SQL

## Etapas do Projeto
1. Criação do banco legado
2. Inserção de dados
3. Criação do banco novo
4. Migração de dados
5. Validação da migração
6. Consultas analíticas
